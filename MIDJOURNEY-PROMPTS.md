## üìù Usage Instructions

1. **Copy the prompt** for the image you want to generate
2. **Paste into Midjourney** (Discord or web interface)
3. **Select the best result** from the 4 options
4. **Upscale** to maximum resolution
5. **Save the image** with the filename specified
6. **Replace the placeholder** in the HTML files

### Optimization Tips:
- Use `--v 6` for the latest Midjourney model
- Add `--style raw` for more photorealistic results
- Add `--q 2` for highest quality (uses 2x GPU minutes)
- Use correct aspect ratios for best results

### Color Scheme Consistency:
- **Primary Blues:** #2563EB, #0EA5E9
- **Accent Colors:** #10B981 (emerald), #F59E0B (amber)
- **Earth Tones:** Warm browns, sandy beiges for cultural images
- **Scientific:** Cool blues, cyans, teals for genetic content
